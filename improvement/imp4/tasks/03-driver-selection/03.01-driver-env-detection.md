## Task Description

Update `src/lib/db/driver.js` to add PostgreSQL as the highest-priority adapter when `POSTGRES_URL` is set. This is the decision point that makes the entire migration opt-in via environment variable.

## Context

The existing `driver.js` has a selection chain that tries Bun SQLite -> better-sqlite3 -> node:sqlite -> sql.js. Each try function is wrapped in try/catch and falls through to the next. PostgreSQL should be inserted as the first attempt, before any SQLite adapter.

## Implementation Steps

### Step 1: Add tryPostgres() function
```js
async function tryPostgres() {
  const url = process.env.POSTGRES_URL;
  if (!url || url.trim() === "") return null;
  try {
    const { createPostgresAdapter } = await import(
      "./adapters/postgresAdapter.js"
    );
    const poolSize = parseInt(process.env.POSTGRES_POOL_SIZE || "20", 10);
    const ssl = process.env.POSTGRES_SSL === "true";
    return await createPostgresAdapter({ connectionString: url, poolSize, ssl });
  } catch (err) {
    console.warn("[driver] PostgreSQL init failed, falling back to SQLite:", err.message);
    return null;
  }
}
```

### Step 2: Update init chain
```js
let adapter = await tryPostgres();
if (!adapter) adapter = await tryBunSqlite();
if (!adapter) adapter = await tryBetterSqlite();
if (!adapter) adapter = await tryNodeSqlite();
if (!adapter) adapter = await trySqlJs();
if (!adapter) throw new Error("No database adapter available");
```

### Step 3: Add env var documentation
Add POSTGRES_URL, POSTGRES_POOL_SIZE, POSTGRES_SSL to `.env.example`:
```
# PostgreSQL connection string (unset to use SQLite)
POSTGRES_URL=postgres://user:pass@localhost:5432/9router
# Connection pool size (default: 20)
POSTGRES_POOL_SIZE=20
# Enable SSL (default: false)
POSTGRES_SSL=false
```

### Step 4: Verify selection logic
```bash
# Without POSTGRES_URL: should pick a SQLite adapter
unset POSTGRES_URL && node -e "require('./src/lib/db/driver.js').getAdapter().then(a => console.log(a.driver))"
# With POSTGRES_URL: should pick postgres
POSTGRES_URL=postgres://test:test@localhost:5432/9router node -e "require('./src/lib/db/driver.js').getAdapter().then(a => console.log(a.driver))"
```
