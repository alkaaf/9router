## Task Description

Create `scripts/migrate-postgres-to-sqlite.js` — the reverse migration script. Reads from PostgreSQL, converts types back to SQLite representation, writes to SQLite. Less commonly needed but essential for safety.

## Context

If a migration to PostgreSQL fails or the operator wants to revert, they need a way to copy the data back. The script is structurally similar to 07.01 but with the source and target swapped, and a few inverse type conversions.

### Step 2: Inverse type conversion
```js
function convertRowBack(row, tableName) {
  const out = { ...row };
  // Booleans: true/false -> 1/0
  for (const col of BOOLEAN_COLUMNS[tableName] || []) {
    if (out[col] !== null) out[col] = out[col] ? 1 : 0;
  }
  // JSON: object -> TEXT
  for (const col of JSON_COLUMNS[tableName] || []) {
    if (out[col] && typeof out[col] === "object") out[col] = JSON.stringify(out[col]);
  }
  // Timestamps: TIMESTAMPTZ -> ISO string
  for (const col of TIMESTAMP_COLUMNS[tableName] || []) {
    if (out[col] instanceof Date) out[col] = out[col].toISOString();
  }
  return out;
}
```
